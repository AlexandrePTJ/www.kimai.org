How to reload Kimai cache

{% highlight plaintext %} bin/console kimai:reload --env=prod {% endhighlight %}

FTP users: please have a look at this documentation.

If you are running an older version of Kimai (before 1.8) you have to use:

{% highlight plaintext %} bin/console cache:clear --env=prod bin/console cache:warmup --env=prod {% endhighlight %}
